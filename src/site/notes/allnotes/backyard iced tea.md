# Ingredients
#ingredients 
* [ ] 2 oz spiced rum
* [ ] 2 oz [[allnotes/lemonade\|lemonade]]
* [ ] 1 oz unsweetened iced tea
* [ ] garnish: lemon wedge

# Directions

fill highball glass with ice.

add rum, lemonade, tea.

stir.

garnish with lemon wedge


# comments
