# Ingredients
#ingredients 
* [ ] 1 oz orgaet
* [ ] 1/2 oz [[allnotes/grenadine\|grenadine]]
* [ ] 1/2 oz lime juice
* [ ] 3 oz coke
* [ ] 2 oz soda water
* [ ] lime or cherry for garnish


# Directions

* mix everything except sodas in a tall glass, stir
* add coke, stir
* add ice, top with soda water, stir
* garnish
